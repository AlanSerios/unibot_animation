# Project Tech Stack

The `uni_animation` project utilizes a highly specific stack of animation libraries to achieve its visual fidelity.

## Required Skills / Libraries
Whenever executing new animations or modifying existing sequences, the following libraries must be leveraged:
- **`pixijs-2d`**: For high-performance 2D rendering and canvas manipulations.
- **`animejs`**: For versatile, timeline-based DOM and SVG keyframe sequences.
- **`lottie-animations`**: For complex vector animations exported from After Effects.
- **`gsap-scrolltrigger`**: For scroll-driven interactions and complex timelines.

These tools are crucial for ensuring the [[Eye Animation Guidelines]] are executed smoothly and efficiently.
